# ESP32 Desktop Environment (OS)

An advanced Desktop Environment for ESP32 based on LVGL v9.

## Features
- **App Manager**: Handle multiple applications within the environment.
- **Window Manager**: Manage application windows.
- **UI Components**: Status bar, Dock, Launcher, and Quick Access Panel.
- **Glassmorphism**: UI utilities for glass effects.
- **Theming**: Integrated theme engine.
- **Connectivity**: WiFi and Bluetooth management.

## Hardware Support
Designed for ESP32-S3 with display support via LVGL and LovyanGFX.

## Getting Started
1. Install ESP-IDF v5.x.
2. Clone this repository.
3. Build and flash:
   ```bash
   idf.py build
   idf.py flash
   ```
